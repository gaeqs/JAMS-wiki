# Instalación

JAMS está disponible en todos los sistemas operativos que soportan Java y JavaFX. Un instalador está disponible
para Windows, Mac y Ubuntu/Debian.

Si prefieres utilizar el jar con todas las dependencias, debes instalar Java 17.
